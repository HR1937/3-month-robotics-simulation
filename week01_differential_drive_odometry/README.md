## How to Run

From inside `week01_differential_drive_odometry/src/`:

### 1. Build and run C++ simulation

```bash
g++ -o odometry_sim main.cpp kinematics.cpp
./odometry_sim
```

Generates:

* `Straight_path.csv`
* `Circular_path.csv`
* `Rotated_path.csv`
> The CSV files are generated by main.cpp and overwritten on each run.
### 2. Plot the paths (Python)

```bash
python graph-plot.py
```

This:

* reads the CSV files
* creates one figure with:
  * left: XY paths,
